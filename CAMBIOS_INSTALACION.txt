═══════════════════════════════════════════════════════════════
  🔄 FLUJO DE INSTALACIÓN NUEVO
═══════════════════════════════════════════════════════════════

ANTES (Manual):
1. Clonar repositorio
2. Copiar config.example.py a config.py
3. Editar config.py manualmente
4. Cambiar PIN en el archivo
5. Instalar dependencias
6. Crear directorios
7. Iniciar panel

AHORA (Automático):
1. Clonar repositorio
2. Ejecutar install.bat o install.sh
3. Ingresar PIN cuando se solicite
4. ¡Listo! Todo configurado automáticamente

═══════════════════════════════════════════════════════════════
  📋 COMANDOS PARA USUARIOS
═══════════════════════════════════════════════════════════════

WINDOWS:
  git clone https://github.com/CristianBatero/panel_web.git
  cd panel_web
  install.bat

LINUX/MAC:
  git clone https://github.com/CristianBatero/panel_web.git
  cd panel_web
  chmod +x install.sh
  ./install.sh
