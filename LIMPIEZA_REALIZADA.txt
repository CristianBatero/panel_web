  PRÓXIMOS PASOS PARA GIT
═══════════════════════════════════════════════════════════════

1. Inicializar repositorio (si no existe):
   git init

2. Añadir archivos:
   git add .

3. Primer commit:
   git commit -m "feat: versión 2.0.0 con diseño iOS y gestión completa"

4. Conectar con GitHub:
   git remote add origin https://github.com/CristianBatero/panel_web.git

5. Subir a GitHub:
   git push -u origin main

6. Crear tags de versión:
   git tag -a v2.0.0 -m "Versión 2.0.0 - Diseño iOS completo"
   git push origin v2.0.0
